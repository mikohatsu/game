## **2\. 핵심 시스템 상세 정의**

### **2.1. 정밀 공정 파라미터 및 조합**

| 파라미터 | 범위 | 기본값 | 용도 |
| :---- | :---- | :---- | :---- |
| **온도 (T)** | 0 \~ 1,000 °C | 500 | 물질 활성화 |
| **압력 (P)** | 0.1 \~ 10.0 atm | 5.0 | 밀도 및 결합 |
| **농도 (C)** | 0 \~ 100 % | 50 | 반응 속도 |

* **비용:** 조합 1회당 **2 AP** 소모.

### **2.2. 품질 등급 및 성공 판정 (등급 C 포함)**

*공식: AvgDev \= (|T오차%| \+ |P오차%| \+ |C오차%|) / 3*

| 등급 | 오차(AvgDev) | 판정 | 판매가 | 오염도 | 비고 |
| :---- | :---- | :---- | :---- | :---- | :---- |
| **S** | \< 1.0% | 대성공 | 2.0배 | \+1 | 기록 갱신 목표 |
| **A** | \< 3.0% | 성공 | 1.5배 | \+3 |  |
| **B** | \< 6.0% | 성공 | 1.0배 | \+5 |  |
| **C** | \< 10.0% | 턱걸이 | 0.5배 | \+10 | 마지노선 |
| **F** | ≥ 10.0% | 실패 | 0.0배 | \+25 | 폐기물 |

### **2.3. 노트(Notebook) 및 저장 시스템 (New)**

플레이어의 모든 활동은 브라우저의 localStorage에 영구 저장됩니다.

#### **A. 실험 일지 (History Tab)**

* **기능:** 플레이어가 시도한 \*\*모든 조합(성공/실패 포함)\*\*을 최근 순으로 나열.  
* **저장 데이터:** \[시도 회차, 사용 재료, 입력값(T/P/C), 결과 아이템, 등급, 편차율\]  
* **UI:** 리스트 형태. 클릭 시 해당 설정값을 조합기에 다시 불러오기(Load) 가능.

#### **B. 연금술 비망록 (Grimoire Tab)**

* **기능:** 성공한 레시피(C등급 이상)만 기록되며, \*\*'최고 기록(Best Attempt)'\*\*이 갱신됨.  
* **갱신 로직:**  
  1. 해당 레시피의 기존 기록이 없으면 저장.  
  2. 기존 기록보다 이번 시도의 등급이 높으면 갱신.  
  3. 등급이 같으면 편차율(AvgDev)이 더 낮으면(정밀하면) 갱신.  
* **표시 정보:** 아이템 아이콘, 이름, **내가 찾은 황금비율(성공 당시 T/P/C)**, 달성 등급.  
* **편의성:** 비망록에서 '세팅하기' 버튼을 누르면 해당 T/P/C 값으로 슬라이더가 자동 세팅됨.

## **6\. 오염도 효과 상세**

* **0\~49 (안정):** 효과 없음.  
* **50\~99 (경고):** 시설 수리비(랜덤 이벤트) 발생 확률 증가.  
* **100\~149 (위험):** 슬라이더 조작 시 1초마다 값이 ±1\~3 랜덤하게 튐.  
* **150\~200 (돌연변이):** '변성 엘릭서' 등 특정 돌연변이 아이템 제작 가능(5% 확률).
